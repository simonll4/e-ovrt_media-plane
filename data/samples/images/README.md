# Mini-dataset DBE — imágenes

Colocar aquí las imágenes del mini-dataset inicial (`.jpg`, `.jpeg`, `.png`).

Recomendaciones para que el set sirva como instrumento de comparación:

- ~30-50 imágenes de obra con personas **con y sin** casco y chaleco.
- Incluir casos fáciles y difíciles: objetos chicos/lejanos, contraluz, oclusión parcial.
- **Congelar el set**: una vez usado en una corrida registrada, no agregar ni quitar imágenes (crear `images_v2/` si hace falta otro set).
- Documentar el origen de cada imagen en este README (fuente, licencia si aplica).

Este directorio aloja los datasets de imágenes descritos en el catálogo `configs/datasets/`, que las run configs (`configs/runs/`) referencian por `source.ref`. Las imágenes no se versionan en git.
